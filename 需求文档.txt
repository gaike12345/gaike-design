盖可设计圈 - 产品需求文档 (PRD)
一、项目概述
1.1 基本信息
应用名称：盖可设计圈
网站类型：品牌展示型
核心定位：面向个人品牌的多元化创意设计与技术服务平台，打造"技能+学习+社交"一体化的设计师成长社区
目标用户：个人品牌（独立设计师、自由创作者、设计学习者、希望建立个人IP的创意从业者）
1.2 技术配置
数据库：使用数据库（需存储作品案例、用户咨询记录、教育咨询信息等）
语言支持：中文
响应式要求：适配移动端/平板/桌面
二、核心功能
2.1 MVP功能（高优先级）
2.1.1 服务展示
展示工作室五大核心业务板块（3D建模、应用开发、原画设计、学习交友、教育咨询），每个板块配有视觉化图标和简短说明

2.1.2 作品集展示
以视觉化网格形式展示代表性作品案例，支持按业务分类筛选，每个作品包含缩略图、标题、简介

2.1.3 联系咨询
提供多种联系方式（微信、邮箱、表单），支持用户提交合作意向或咨询请求

2.2 增强功能（中优先级）
2.2.1 关于工作室
介绍工作室理念、团队背景、核心竞争力，建立信任感和品牌认同

2.2.2 学习社区入口
展示学习交友和教育咨询服务的相关信息，引导用户加入社群或预约咨询

2.2.3 动态/博客
发布工作室动态、设计教程、行业洞察等内容，增强用户粘性和专业形象

2.3 扩展功能（低优先级）
2.3.1 用户案例见证
展示客户评价、合作案例详情、前后对比等，增强说服力

2.3.2 在线预约系统
支持用户在线预约咨询时间、选择服务类型、填写需求详情

三、页面结构
3.1 首页布局
整体布局模式：非对称创意布局 + Bento Grid混合

选择理由：

目标用户为个人品牌和创意从业者，需要传递"创意、个性、不拘一格"的品牌气质
多元化业务（3D建模、应用开发、原画、教育、社交）需要用Bento Grid进行多维展示
打破传统企业官网的严肃感，营造艺术创意氛围
核心区域：

Header（顶部导航）

左侧：Logo（"盖可设计圈"品牌标识）
中部：导航菜单（首页、服务项目、作品集、关于我们、学习社区、联系我们）
右侧：CTA按钮（"立即咨询"）
移动端：汉堡菜单
Hero区（首屏）⚠️ 关键视觉区域

背景实现方式：方案D - 渐变色背景 + 3D装饰元素

选择理由：

工作室业务涵盖3D建模、应用开发等科技感强的领域，渐变色传递前沿和创新
目标用户为年轻创意人群，需要视觉冲击力和记忆点
无单一实体产品展示，适合用抽象视觉传递品牌调性
具体实现：

背景：液态渐变网格（Gradient Mesh），蓝紫色系流动效果
前景装饰：悬浮3D几何图形（立方体、球体、抽象造型），轻微旋转动画
大标题：超大字体排版（80-96px），粗体无衬线
副标题：简短有力的价值主张
CTA按钮：高对比度强调色，hover有光晕效果
前景内容：

核心价值主张（大标题）："让创意落地，让设计发声"
简短说明（副标题）："3D建模 | 应用开发 | 原画设计 | 学习成长 | 创意社交"
行动按钮（CTA）："探索服务" / "加入社群"
设计意图：通过未来感的渐变背景和3D元素，让用户立刻感受到工作室的创新能力和多元业务特色，吸引创意人群的关注和共鸣

业务展示区（Bento Grid布局）

采用不规则网格展示5大业务板块
卡片大小差异化：核心业务（3D建模、原画设计）用大卡片（2:1比例），其他用标准卡片（1:1）
每个卡片：渐变图标 + 业务名称 + 简短描述 + hover微交互（轻微上浮+阴影加深）
视觉风格：Glassmorphism毛玻璃效果卡片，半透明背景+模糊滤镜+细边框高光
精选作品区

展示方式：非对称网格（3-4个代表作品）
每个作品：大图缩略图 + 标题 + 业务标签 + 查看详情链接
交互：hover时图片放大1.05倍，显示"查看案例"按钮
品牌理念区

左右分栏布局：左侧文字（工作室理念），右侧视觉（团队工作场景或抽象创意图）
文案精炼有力，传递"创意+技术+成长"的核心价值观
行动号召区

深色背景 + 居中布局
大标题："准备好开启你的创意之旅了吗？"
CTA按钮："立即咨询" / "加入学习社群"
Footer（底部信息）

联系方式（微信、邮箱、社交媒体）
快速导航链接
版权信息
3.2 关键内页
页面：服务项目
页面作用：让用户深入了解每项服务的具体内容、流程、适用场景，促进转化

主要内容区域：

页面Hero区：大标题"我们的服务"，简短说明"从概念到落地，全方位支持你的创意项目"
服务详情页块（5个板块，垂直排列）：
3D建模：服务内容（角色建模、场景建模、产品可视化等）、应用场景、技术工具、案例链接
应用开发：服务内容（移动应用、Web应用、交互原型等）、技术栈、开发流程、案例链接

学习交友：服务内容（设计课程、作品指导、社群交流、线下活动等）、适合人群、社群福利

转化设计：每个服务板块底部放置"预约咨询"按钮，页面底部放置"获取定制方案"CTA
页面：作品集
页面作用：展示工作室实力和设计风格，建立专业信任，激发合作意向

主要内容区域：

筛选导航：按业务类型筛选（全部/3D建模/应用开发/原画设计），Tab切换
作品网格：Bento Grid布局，卡片大小根据作品重要性差异化
作品详情弹窗/页面：点击作品后展示大图、项目背景、设计思路、技术实现、客户反馈
转化设计：作品详情页底部放置"类似项目咨询"按钮
页面：关于我们
页面作用：建立情感连接，传递品牌故事和价值观，增强信任感

主要内容区域：

品牌故事：工作室创立初衷、发展历程、核心理念
团队介绍：核心成员照片、专业背景、擅长领域（突出多元化和专业性）
核心竞争力：为什么选择我们（如：跨领域能力、个性化服务、持续学习文化等）
价值观展示：用视觉化方式呈现"创意、技术、成长、连接"等关键词
转化设计：页面底部放置"加入我们"或"合作咨询"CTA
页面：学习社区
页面作用：吸引设计学习者加入社群，推广教育咨询服务

主要内容区域：

社群价值主张：加入你能获得什么（技能提升、人脉资源、项目机会等）
社群活动展示：线上分享会、作品点评、线下聚会、联合创作等
学员见证：学员成长故事、作品进步对比、感言评价
课程/服务介绍：设计课程大纲、教育咨询服务详情、收费标准（可选公开）
转化设计：醒目的"加入社群"按钮、"预约咨询"表单
页面：联系我们
页面作用：提供多种联系渠道，降低用户咨询门槛，促进转化

主要内容区域：

联系信息：微信二维码、邮箱、电话（可选）、社交媒体链接
咨询表单：姓名、联系方式、服务类型（下拉选择）、项目描述（文本域）、提交按钮
工作流程：简要说明合作流程（咨询→需求沟通→方案报价→执行→交付）
FAQ：常见问题解答（如：服务周期、收费方式、修改次数等）
转化设计：表单提交后显示成功提示，并引导添加微信进一步沟通
四、核心交互流程
4.1 流程：用户咨询流程
操作步骤：

用户在任意页面点击"立即咨询"或"预约咨询"按钮
弹出咨询表单（或跳转到联系页面），包含字段：
姓名（必填）
联系方式（微信/手机/邮箱，必填）
服务类型（下拉选择：3D建模/应用开发/原画设计/学习交友/教育咨询/其他，必填）
项目描述（文本域，选填）
预算范围（可选）
用户提交 → 显示loading动画 → API调用
成功：显示成功提示（"感谢提交！我们将在24小时内与您联系"），同时显示微信二维码引导添加
失败：显示错误信息（如网络问题），允许重试
异常处理：

网络超时：提示"网络连接不稳定，请检查后重试"
服务器错误：提示"系统繁忙，请稍后再试或直接添加微信联系"
必填项未填：实时验证，红色提示框标注
4.2 其他关键流程
作品浏览流程：

用户进入作品集页面
可按业务类型筛选（点击Tab）
点击感兴趣的作品缩略图
弹出作品详情（或跳转详情页），展示大图、项目背景、设计思路
用户可点击"类似项目咨询"按钮发起咨询
社群加入流程：

用户进入学习社区页面
了解社群价值和活动
点击"加入社群"按钮
显示微信群二维码或填写入群申请表
管理员审核后邀请入群
五、视觉与品牌策略
5.1 品牌个性定位
核心定位：年轻创新型 + 艺术创意型

目标情感：用户访问网站应该产生的3个核心感受

第一层感受：创意感（打破常规、充满想象力、艺术气息）
第二层感受：专业信任（技术实力强、案例丰富、可靠）
第三层感受：成长激励（学习资源丰富、社群氛围好、能提升自己）
差异化策略：与同行业其他品牌的差异点

我们不是传统设计公司（只接项目、冷冰冰的甲乙方关系），而是创意成长伙伴（项目合作+学习成长+社群连接）
我们不是单一技能工作室（只做3D或只做原画），而是跨领域综合平台（3D+开发+原画+教育+社交，一站式解决）
核心差异点："技能+学习+社交"三位一体，既提供服务也培养人才，既做项目也建社群
设计决策摘要：

主色：蓝紫渐变 #4A5BFF → #7B3FF2 | 辅色：深空黑 #0A0A0F | 强调色：霓虹青 #00F5FF
布局模式：非对称创意布局 + Bento Grid混合
动效基调：动感科技 + 创意趣味（3D悬浮、光标交互、视差滚动）
字体策略：超大无衬线粗体标题（80-96px）+ 常规正文（16-18px）
采用趋势：Gradient Mesh液态渐变背景 + Glassmorphism毛玻璃卡片 + Scroll-driven动画 + 自定义光标交互
5.2 设计语言（基于目标情感推导）
整体风格：赛博创意风 - 融合科技感与艺术性的前卫视觉风格

核心调性：

创意感 → 大胆用色、非对称布局、3D装饰元素
专业信任 → 清晰的信息架构、高质量作品展示、数据化呈现
成长激励 → 温暖的人文元素、社群活动照片、学员成长故事
设计目标：通过未来感的视觉语言和创意交互，让年轻创意人群产生"这就是我要找的工作室"的强烈认同感，同时通过专业案例和清晰服务建立信任

5.3 色彩方案（含推导理由）
主背景色：深空黑 #0A0A0F - 深色背景传递专注、沉浸、高端感，符合创意人群对"酷"和"专业"的双重追求，同时让渐变和3D元素更突出

主色调：蓝紫渐变 #4A5BFF → #7B3FF2 - 冷色系渐变传递科技感、创新、未来，蓝色代表专业信任，紫色代表创意艺术，渐变效果增加动感和流动性

强调色：霓虹青 #00F5FF - 高饱和度的青色作为点缀，用于CTA按钮、重要链接、hover效果，制造视觉焦点和行动引导，传递活力和前沿感

辅助色：

暖橙 #FF9A56 - 用于学习社区相关元素，传递温暖、成长、活力，平衡冷色调的距离感
纯白 #FFFFFF - 主要文字颜色，确保在深色背景上的可读性
文字色：

深色背景时：纯白 #FFFFFF（主标题）、浅灰 #E0E0E0（正文）
浅色背景时（如卡片内部）：深灰 #1A1A1A（标题）、中灰 #4A4A4A（正文）
图片/图标风格：

作品图片：高质量渲染图，保持原始色彩，不添加滤镜
业务图标：3D立体图标或渐变线性图标，与整体风格统一
团队/场景图片：真实工作场景照片，适当调色偏冷色调，保持专业感
装饰元素：抽象3D几何图形、粒子效果、光晕线条
5.4 布局策略（含推导理由）
首屏布局：非对称英雄区 - 文字区占40%（左侧），视觉区占60%（右侧3D装饰），打破对称传递创意和不拘一格，符合目标用户的审美偏好

内容区布局：Bento Grid不规则网格 - 用于业务展示和作品集，卡片大小差异化（1:1、2:1、1:2混合），传递信息丰富但不杂乱、现代科技感的视觉印象

留白策略：平衡留白 - 不过度留白（避免显得空洞），也不密集堆砌（保持呼吸感），区块之间间距80-120px，卡片内部间距24-32px

排版原则：

超大标题传递力量感和自信（H1: 80-96px，H2: 48px，H3: 32px）
字重对比明显（标题Bold 700，正文Regular 400）
行间距宽松（1.6-1.8倍），提升可读性
关键信息用强调色或加粗突出
5.5 字体规范（基于品牌气质）
标题字体：

样式：粗体无衬线（如思源黑体 Bold、Inter Bold、Montserrat Bold）- 传递现代感、力量感、科技感
大小：H1 80-96px（桌面端）/ 40-48px（移动端）| H2 48px / 32px | H3 32px / 24px
字间距：略微收紧（-0.02em），增强整体感
正文字体：

样式：常规无衬线（如思源黑体 Regular、Inter Regular）- 清晰易读、现代简洁
大小：16-18px（桌面端）/ 14-16px（移动端）
行高：1.6-1.8倍
小字/辅助文字：

大小：14px（桌面端）/ 12px（移动端）
颜色：浅灰 #CCCCCC，用于注释、标签、次要信息
特殊场景：

英文数字可用等宽字体（如JetBrains Mono）点缀，增强科技感
引用文案可用斜体或不同字重区分
5.6 交互动效（含推导理由）
整体基调：动感科技 + 创意趣味 - 符合年轻创意人群的审美，传递创新和活力，同时保持专业度不轻浮

核心动效（基于品牌个性选择4个关键动效）：

Gradient Mesh液态渐变背景动画

效果：Hero区背景的渐变色缓慢流动变化，像液体一样柔和过渡
参数：动画时长15-20秒循环，多色点柔和移动
理由：传递未来感和流动性，增强视觉吸引力
3D几何图形悬浮旋转

效果：Hero区和业务卡片的3D装饰元素轻微悬浮（上下浮动3-5px）+ 缓慢旋转
参数：悬浮动画2-3秒循环，旋转速度5-10度/秒
理由：增强空间感和科技感，呼应3D建模业务
Scroll-driven淡入动画

效果：页面滚动时，内容区块依次淡入 + 向上位移（translateY 30px → 0）
参数：元素进入视口20%时触发，过渡时长600ms，缓动函数cubic-bezier(0.4, 0, 0.2, 1)
理由：创造沉浸式滚动体验，引导用户视线，增强叙事节奏
Glassmorphism卡片hover微交互

效果：鼠标悬停时，毛玻璃卡片轻微上浮（4px）+ 阴影加深 + 边框高亮
参数：过渡时长300ms，缓动函数ease-out，阴影从0 4px 20px rgba(0,0,0,0.1)变为0 8px 40px rgba(74,91,255,0.3)
理由：提供清晰的交互反馈，增强品质感和细节打磨印象
辅助动效：

自定义光标：圆形光标 + hover可点击元素时放大1.5倍 + 变色为强调色（仅桌面端）
按钮光晕效果：CTA按钮hover时，边缘出现霓虹青光晕扩散动画
数字滚动动画：如有数据展示（如"服务100+客户"），数字从0滚动到目标值
动效参数规范：

过渡时长：快速反馈200-300ms（按钮hover），内容切换400-600ms（区块淡入），页面级过渡600-800ms
缓动函数：克制型用cubic-bezier(0.4, 0, 0.2, 1)，弹性型用cubic-bezier(0.34, 1.56, 0.64, 1)
滚动触发：元素进入视口20%时开始动画
hover效果：位移幅度2-6px，缩放范围1.02-1.05
交错延迟：列表项依次进入时每项间隔50-100ms
动效性能原则：

避免过度动画导致卡顿，尤其是移动端
移动端简化动效（关闭3D旋转、减少粒子效果），优先考虑性能
关键转化路径上的动效不能干扰用户操作（如表单提交流程保持简洁）
5.7 品牌元素
Logo使用：

位置：导航栏左侧、页脚、加载动画、favicon
规范：深色背景用白色版本，浅色背景用深色版本
最小尺寸：高度不小于32px，确保清晰度
品牌视觉符号：

3D几何图形：立方体、球体、圆锥体等抽象造型，作为装饰元素贯穿全站
渐变光晕线条：流动的曲线或光带，增强科技感和动感
粒子效果：轻微的粒子漂浮或汇聚效果，用于Hero区或加载动画
品牌色彩系统：

确保所有页面严格遵循主色、辅色、强调色的使用规范
禁止使用未经定义的第三方颜色
5.8 文案调性（含示例）
语言风格：友好口语 + 创意表达 - 既保持专业度又拉近距离，用年轻人喜欢的语言风格，避免过于正式和距离感

六、多端适配策略
6.1 响应式要求
网站必须在移动端/平板/桌面端提供良好体验，确保创意视觉效果在不同设备上都能正常呈现，同时保证性能和可用性。

6.2 断点定义
设备类型	屏幕宽度	布局特征
移动端	< 768px	单列，导航折叠为汉堡菜单，标题字号缩小，简化3D动效
平板	768px - 1023px	2-3列布局，导航可展开，保留适度动效
桌面端	≥ 1024px	多列布局，完整导航，丰富交互和3D效果
6.3 核心适配规则
导航栏：

移动端：汉堡菜单（右上角），点击后全屏或侧边滑出
平板：可扩展菜单或简化版水平菜单
桌面端：完整水平菜单 + CTA按钮
Hero区：

移动端：单列垂直布局，标题40-48px，3D装饰元素简化或静态化
平板：左右分栏，标题56-64px，保留轻度3D效果
桌面端：非对称布局（40:60），标题80-96px，完整3D动画和渐变背景
业务展示区（Bento Grid）：

移动端：单列垂直排列，卡片1:1比例
平板：2列网格，部分卡片保持2:1比例
桌面端：3-4列不规则网格，充分展现Bento布局特色
作品集：

移动端：单列，图片宽度100%
平板：2列网格
桌面端：3-4列Bento Grid，hover效果完整
联系表单：

移动端：单列，输入框全宽，按钮全宽
桌面端：可适当增加表单宽度，提升填写体验
性能优化：

移动端：
图片懒加载（lazy load）
关闭或简化3D旋转、粒子效果
渐变背景用静态图片或简化版本
减少同时播放的动画数量
桌面端：
高分辨率图片
完整3D效果和动效
可加载更多装饰元素
触控优化：

移动端和平板：
按钮和可点击区域最小尺寸44x44px
移除hover效果（改为点击反馈）
滑动切换代替悬停展示
七、补充说明
特殊要求：

3D元素实现：建议使用Three.js或Spline等3D库实现Hero区的3D几何图形，确保性能和兼容性
渐变背景性能：Gradient Mesh液态渐变需注意性能，移动端可降级为静态渐变图片
作品图片质量：作品集图片需保证高质量，建议提供@2x和@3x版本适配高分辨率屏幕
社群二维码：微信二维码需定期更新，建议后台可配置或使用动态码
